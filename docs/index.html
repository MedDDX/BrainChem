Chemistry Draw: GitHub Pages Gallery

This static site is served from the docs/ folder and reads chemical data from docs/chemicals/. Edit the JSON in the repository to add more examples, then push to deploy with GitHub Pages.

How to extend the site

  1. Edit docs/chemicals/chemicals.json to add new entries.
  2. Optional: use draw_smiles.py to generate new images and commit them into docs/chemicals/ for static hosting.
  3. Push to the repository. The GitHub Pages workflow will deploy automatically.

Each card links the SMILES string so you can quickly run python draw_smiles.py <smiles> to generate a PNG locally.

Built for GitHub Pages using only static assets from this repository.
